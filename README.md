<h1>Pokédex</h1>

<h5>This project uses:</h5>
<ul>
    <li>React</li>
    <li>Redux</li>
    <li>Material UI</li>
    <li>Pokemon API: https://pokemontcg.io/</li>
</ul>

<h3>To run this app:</h3>
<h5>Install dependencies by:</h5>
<li>npm install</li>
<h5>and start by:</h5>
<li>npm start</li>
